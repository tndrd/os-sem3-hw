# os-sem3-hw
Homeworks for Operating Systems course held in MIPT (Semester 3).

# How to build
```bash
git clone https://github.com/tndrd/os-sem3-hw.git --recursive
cd os-sem3-hw
mkdir build
cd build
cmake ..
cmake --build .
```
> The executables are placed into ```/build/<PROJECT_NAME>/``` directory.
