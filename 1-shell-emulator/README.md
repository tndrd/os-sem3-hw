# Task 1: Shell Emulator
The goal is to create emulator of Linux shell. The shell should support I/O redirect using ```>``` or ```<``` operators, and command pipeline with ```|``` operator.

# How to run
```bash
cd build/1-shell-emulator
./1-shell-emulator
```

# How to test

```bash
cd build/1-shell-emulator
./RunTests
```
